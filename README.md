# homebrew-tap

Homebrew casks for my macOS apps. All builds are signed and notarized.

```bash
# Harbor — boot all your local dev servers with one button, MCP-native
brew install --cask luke-fairbanks/tap/harbor

# Battery Hog — see what's draining your battery, 100% local
brew install --cask luke-fairbanks/tap/battery-hog
```
